Cookie policy

This page describes the technical trackers and external resources that may be used on the TORFIX website. It should be updated if analytics, advertising or additional marketing tools are added later on.

What is a cookie?

A cookie is a small file stored on your device when you visit a website. It can be used to make the site function properly, remember preferences, measure traffic or enable third-party services.

Current usage on TORFIX

At the time of writing, the TORFIX website mainly relies on static content, visual resources and a contact form. No advertising or advanced tracking service is described here unless it is actually enabled on the live website.

Strictly necessary cookies

Some technical operations may require strictly necessary requests in order to display the website correctly or transmit the contact form. These technical traces are not intended to profile users but to make the service work as expected.

External resources currently used
Google Fonts

The website loads certain fonts from Google Fonts. During this request, technical data such as IP address or browser details may be processed by the provider to deliver the requested resources.

Formspree

The website contact form is connected to Formspree for technical message routing. Using this service may involve processing technical information necessary to transmit the enquiry.

Vercel hosting

The website is deployed on Vercel infrastructure. Technical logs may exist for availability, security and infrastructure maintenance purposes.

User control
Browser settings

You can configure your browser to block, limit or delete cookies. Doing so may however affect the normal operation of parts of the website or its contact form.

Future updates

If TORFIX later adds analytics, marketing chat tools, advertising pixels or any consent-based script, this page should be updated and a suitable consent mechanism should be introduced where required.

Contact

For any question about cookies and trackers used on the TORFIX website, please contact info@torfix.be.
